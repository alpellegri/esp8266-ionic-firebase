# uHome

a **Cloud Home Automation** based on:
* Firbase + FCM
* ESP8266 / ESP32

Please note that this is in **Alpha status**. This project is **under heavy development, experimental, unversioned and not stable**.

## **Features**
* Cloud (Firebase)
* Flutter
* DTH22
* RF433
* FOTA

## Install
Install platformio

### ESP8266
Build esp8266 folder

### ESP32
Build esp32 folder
